# 바로GO — 전국 방문 마사지 안내 (네비게이션 프로토타입)

전국 출장마사지 예약 안내 플랫폼의 상단 네비게이션 및 메인 헤더 프로토타입입니다.
명세서 기준 7개 상단 메뉴와 메가 메뉴, 모바일 토글, 푸터 사업자 정보를 포함합니다.

## 파일 구조

```
.
├── index.html              # 메인 페이지 (헤더/네비/히어로/푸터)
├── assets/
│   ├── css/style.css       # 레이아웃, 네비게이션, 메가 메뉴, 반응형 스타일
│   └── js/nav.js           # 메가 메뉴 hover/click 토글, 모바일 햄버거, ESC 처리
└── README.md
```

## 상단 메뉴 구조

1. 홈 — `/`
2. 지역별 찾기 — `/area/{시·도}/` (17개 광역 지자체)
3. 서비스 안내 — `/service/...` (출장마사지·홈타이·스웨디시·아로마·스포츠·커플·호텔·기업 방문)
4. 예약 안내 — `/reservation/...` (예약 방법·가격·이용 전 확인·취소 환불·결제)
5. 후기·정보 — `/review/...` 및 `/guide/...` (2열 메가 메뉴)
6. 고객센터 — `/support/...` (공지·FAQ·문의·제휴·불편 신고)
7. 바로GO 소개 — `/about/...` (브랜드·운영 원칙·관리사 기준·안전 정책·개인정보·약관)
